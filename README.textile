h1. tween.lua demo

This is a demo of the tween.lua library. It shows some of the transformations that can be done with it.

The demo requires "LÖVE":http://love2d.org

h2. Instructions

* Download and install LÖVE
* Download/uncompress this repo to a folder
* Execute with LÖVE
<pre>
cd folder/where/this/repo/is
love .
</pre>
